A running list of the keybindings I actually reach for when I'm driving Claude Code day to day. Most of these became muscle memory without me noticing; writing them down is mostly an exercise in paying attention.

- `Esc` — interrupt the current response. The one I use most.
- `Shift+Tab` — cycle through plan / auto-accept / default modes without leaving the keyboard.
- `Ctrl+R` — expand the last tool call's output when I want to see what actually ran.
- `Up arrow` — scroll back through recent prompts to tweak and resend.
- `Ctrl+L` — clear the visible transcript when the scrollback gets noisy.
- `Cmd+K` — fuzzy-open a file from the project into the conversation context.
- `Ctrl+C` twice — bail out of the session when something is truly wedged.

I'll add to this as I notice new ones sneaking into my flow.
